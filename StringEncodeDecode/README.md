# Encode and Decode a string
Translate String array to a String and vice versa.
The strings may contain numbers, blanks, special chars etc.
eg:  String Encode/Decode
```
input:
[“hello” , “world”, "!", ""]
input:
[“1hello” , “world”, “!”, ’””, “hi” , “”]
```
Encode -> to String
Decode -> back to list
